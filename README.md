# taiwan-weather CLI
A CLI for [taiwan-weather](https://www.npmjs.com/package/taiwan-weather) module.

## Installation

`npm install taiwan-weather-cli`\

## Usage

With default values:

`twwt --api-key <API_KEY>`


### Why this name?
`twwt` stands for `t`ai`w`an-`w`ea`t`her.

It's just quicker to type.
